Controls:

Player1
-------

Move Left               - Left arrow
Move Right              - Right arrow
Drop                    - Down arrow
Rotate Clockwise        - Right Control/Right Option
Rotate CounterClockwise - Right Shift/Right Command

... or gamepad #1

Player2
-------

Move Left               - A
Move Right              - D
Drop                    - S
Rotate Clockwise        - E
Rotate CounterClockwise - Q

... or gamepad #2
